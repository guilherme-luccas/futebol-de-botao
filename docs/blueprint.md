# **App Name**: Button Football Tournament

## Core Features:

- Player Input: Input field to add player names, accommodating a variable number of players (e.g., 5, 6, 9).
- Game Scheduling: Generates the match schedule such that everyone plays against each other, optimized to minimize wasted field capacity. A 'tool' chooses when and if to implement specific matches or assign players to fields, and handles byes for an odd number of players.
- Field Input: Input field to specify the number of available football fields for matches.
- Match Display: Display matches on a table, assigning players to fields and indicating who is out in each round if an odd number of players exist.
- Point Calculation: Automatically calculates points based on match results (Win = 3, Draw = 1, Loss = 0).
- Ranking Display: Show player rankings, including wins, draws, losses, total points, and tie-breakers (most wins, then goal difference).
- Playoff Bracket Generation: After the initial tournament, generates the playoff bracket: 1st vs 4th, 2nd vs 3rd, semi-finals, and final.

## Style Guidelines:

- Primary color: Vibrant purple (#800080) for a futuristic and energetic feel.
- Background color: Light gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Bright green (#00FF00) for highlighting important elements like the currently-selected players, and for buttons.
- Body and headline font: 'PT Sans' for a modern and readable experience.
- Use sports-themed icons, for example, icons showing players.
- Responsive layout to ensure seamless experience across different devices (mobile, tablet, desktop).
- Smooth transitions and animations for loading matches and displaying results.
